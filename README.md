# NotePulse â€“ MERN Stack Note-Taking Application

A full-stack note management application built using the **MERN stack** with secure API practices and a modern responsive UI.

## Features

* Full CRUD functionality (Create, Read, Update, Delete)
* RESTful API built with Express.js
* MongoDB Atlas cloud database integration
* API Rate Limiting using Upstash Redis (HTTP 429 protection)
* Responsive frontend built with React, Vite, and Tailwind CSS
* Optimistic UI updates for improved user experience

---

## Tech Stack

**Frontend:** React, Vite, Tailwind CSS, React Router, Axios
**Backend:** Node.js, Express.js
**Database:** MongoDB Atlas
**Security:** Upstash Redis (Rate Limiting)

## Key Learning Outcomes

* Designed modular backend architecture (routes, controllers, middleware)
* Implemented secure API rate limiting
* Managed frontend state and routing efficiently
